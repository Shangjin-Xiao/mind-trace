# 心迹 Mind Trace

一款简洁的日记应用，记录您的心路历程。

## 理念

- 「心」：对一言的感悟，内心的共鸣
- 「迹」：个人的随记轨迹，生命的印记

## 功能特点

- 每日记录想法和感悟
- 简约优雅的界面设计
- 支持本地存储
- Material Design 3 界面设计
- 支持移动端和桌面端

## 开始使用

1. 确保已安装 Flutter SDK
2. 克隆项目：
